# 安装&使用

::: tip 提示

@itpeilibo/p-ui 是基于 vue2 + Element-ui 和 ant-design-vue 二次封装的基础组件

:::

## Install

```js
npm install @itpeilibo/p-ui -S
&
yarn add @itpeilibo/p-ui -S
```

## Usage

```js
// main.js
import Pui from "@itpeilibo/p-ui"

Vue.use(Pui)
```
